
Der Ordner "graphics" enthält die verwendeten Grafiken für das Tastaturlayout Neo_2 ADNW und KOY als svg (scalable vector graphics).

Diese können mit einem vector graphics editor wie z.B. Inkscape editiert werden, um zum Beispiel ein anderes Layout zu erzeugen.

Anschließend kann jede Ebene als png (portable_network_graphics) im Format 1000 x 250 Pixel exportiert und unter /assets/<layoutname>/ abgespeichert werden.

Das für die Anordnung der Buchstaben auf der Bildschirmtastatur zuständige Key Array befindet sich am Ende der Datei "key-overlay.vala".
